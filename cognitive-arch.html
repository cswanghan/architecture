🎓 学生正常提交数学作业 (无感知) →
// 被动数据 → 自动诊断 认知引擎: 自动错题归因 (L1) 知识追踪: 定位"空间几何辅助线"盲区 学生世界模型自动更新
⇅ 交叉点: 学情数据 → 喂入教师 AI 工具 ⇅
// 教师主动使用 AI (两大方向) 方向A: AI生成差异化教案 + 分层练习 方向B: 学情分析 — 8人掉队 (共性盲区) 方向B+: 课堂世界模型模拟分三组效果
→ 教师审核: 选择分组方案 + 确认教案 →
// 课堂实施 → 新一轮数据 学生侧: 完成老师布置的分层练习 (被动) 教师侧: 差异化课堂实施 (主动)
→ 数据回流 → 模型更准 → 下一轮螺旋上升
